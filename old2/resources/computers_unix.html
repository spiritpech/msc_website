About Unix Machines
  
Computers in MSC Laboratory

The MSC Lab has 5 UNIX machines and 3 Windows machines. The unix machines are labelled mechatro1.ME - mechatro5.ME. The Windows machines are labeled MSC PII, MSC K6 and ...

 
File Server and Home Directories

Mechatro2 acts as the file server. This means that all files of your home directory are present in the hard drive of mechatro2. Other unix machines access this information over the network.

 
Using the Floppy and Zip Drive

This is what you need to do to use the zip drive on mechatro2.

  • cd (This will take you to your home directory).
  • Put your zip disk into the zip drive
  • mount /mnt/zip;cd /mnt/zip The directory /mnt/zip will now contain the contents of the zip disk. You can now do the same operations that you would with any other file in your home directory

    •  
  • IMPORTANT: Please unmount the zip drive after you are done using it. This is important because other users will not be able to use the zip drive since you would have control over it for as long as it is mounted. The procedure to do this is:
  • cd
  • umount /mnt/zip

    •  
    To use the floppy drive in the linux machines,
  • cd
  • Put your floppy into the floppy drive
  • mount /mnt/floppy;cd /mnt/floppy The directory /mnt/floppy will now contain the contents of the floppy disk. You can now do the same operations that you would with any other file in your home directory

    •  
  • IMPORTANT: Again, please unmount the floppy drive after you are done using it. This is important because other users will not be able to use the zip drive since you would have control over it for as long as it is mounted. The procedure to do this is:
  • cd
  • umount /mnt/floppy

    The Web Server and Personal Home Pages

    Mechatro2 acts as the web server as well. The www address is: http://mechatro2.ME.Berkeley.EDU. For information on how to create a personal home page, click here.

    To publish your html documents on the mechatro2 web server, do the following.

    % cd
    % mkdir public_html

    Put your html document structure in public_html. Name your home page index.html.

    Then http://mechatro2.ME.Berkeley.EDU/~your_login_name will point to ~/public_html/index.html

     
    Mail server

    Mechatro2 acts as the labs mail server. Each member of the lab has a mailbox in mechatro2 which receives email. A mail program (which is easy and convenient) you could use is pine. Type pine at the command prompt to access your mailbox.

    For convenience, mechatro2 also has group mailing lists. The available mailing lists are:

    tomi_group :All current members of Prof. Tomizuka's research group.
    avcs :Group mailing list for the Vehicle Control Group.
    alumni :Group mailing list for Alumni.

    e.g.: If you wish to mail tomi_group, send your mail to tomi_group@mechatro2.me.berkeley.edu
