About Windows Machines
  

The MSC Lab has 3 Windows machines.
 
The windows machines are labelled MSC P2, MSC K6 and P-133.
 
P2 and K6 run Windows NT and P-133 runs Windows 98. Please ask for an account on these machines. You could contact Shashi or Erwin directly or mail wizard@mechatro2.me.berkeley.edu.

 
File Server and Home Directories

MSC P2 acts as the file server for the Windows NT machines(namely P2 and K6). This means that all files of your home directory are present in the hard drive of MSC P2. You can however access the information from K6 as well by clicking on the home directory icon.
